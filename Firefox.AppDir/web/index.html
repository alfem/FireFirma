Lo técnico

¿Cómo funciona?

FireFirma es una recopilación de los elementos necesarios para firmar (navegador, java, aplicación de firma, certificados raíz, scripts, librerías, etc), empaquetados en un sólo fichero ejecutable.

Este fichero -bastante gordo- es independiente de la versión de linux que utilices (teóricamente, al menos).

Para aislar estos elementos del resto del sistema, utilizamos el formato AppImage. Básicamente es un directorio con las aplicaciones, runtimes, librerías... dispuesto según unas especificaciones concretas. Se comprime y se le pega un pequeño programa que es el que finalmente hará la descompresión, preparará el entorno y lanzará la aplicación cuando el usuario lo ejecute.
